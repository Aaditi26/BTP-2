Problem Statement

Research Framing

The work addresses intelligent process planning for machining by integrating a data-driven predictor with a multi-objective evolutionary optimizer. The system identifies the best possible operation sequence and machining parameters for a job, while accounting for practical manufacturing constraints and competing objectives such as time, cost, and tool wear.

Objective Functions

Optimization Goals

Primary Objective

Minimize Total Machining Time

The optimizer prioritizes process plans that complete the full machining job in the least time.

Secondary Objectives

  • Minimize Total Cost
  • Minimize Tool Wear
  • Maximize Surface Finish Quality

Input Variables

Given Inputs and Decision Variables

A. Given Inputs

  • Job ID
  • Material Type and Hardness
  • Workpiece Geometry
  • Tolerance Requirements
  • Surface Finish Requirement
  • Machine Type
  • Available Operations List

B. Decision Variables

Sequence Variables

S = [O1, O2, O3, ..., On]

Machining Parameters for each operation

  • Cutting Speed (Vi)
  • Feed Rate (fi)
  • Depth of Cut (di)
  • Tool Type (Ti)
  • Coolant Condition (Ci)

Output Variables

Prediction and Optimization Results

Random Forest Predictions

  • Machining Time per operation
  • Operation Cost
  • Surface Roughness

Final Optimization Output

  • Optimal Sequence S*
  • Optimal Parameters for each operation
  • Minimum Total Machining Time
  • Pareto Optimal Solution Set

Constraints

Feasibility Rules

Precedence Constraints

  • Facing must be first
  • Centering before Drilling
  • Finishing must be last

Machine Constraints

Vmin ≤ Vi ≤ Vmax
fmin ≤ fi ≤ fmax
dmin ≤ di ≤ dmax

Material Constraints

For Steel:     V ≤ 200 m/min
For Aluminum:  V ≤ 400 m/min

Tool and Feasibility Constraints

  • Only valid tools per operation
  • Respect tool life limitations
  • Use all operations exactly once
  • No duplicate sequence entries

Mathematical Formulation

Objective Functions

Single Objective

Minimize:
Total Time = Σ Ti

Each Ti is predicted by the Random Forest model from operation and parameter inputs.

Multi-Objective NSGA-II

Minimize:
F1 = Σ Ti   (Time)
F2 = Σ Ci   (Cost)
F3 = Σ Wi   (Tool Wear)
